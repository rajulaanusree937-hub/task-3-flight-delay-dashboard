# Flight Delay Dashboard Using Power BI

## Overview

This project is an interactive dashboard created using Power BI to analyze flight delay data and visualize airline performance.

## Dataset

Flight Data 2024

## Tools Used

* Power BI
* Flight Data 2024 Dataset

## Dashboard Features

* Total Flights
* Average Arrival Delay
* Flights by Airline
* Average Delay by Airline
* Top Destination Airports
* Monthly Flight Trends
* Interactive Filters
